# OEM Reference Documents

This directory records vendor/OEM PDFs used during hardware bring-up and display-driver investigation. The PDFs themselves are not checked into the public tree; download them from the vendor before doing driver work, and keep local copies out of Git unless redistribution rights are confirmed.

Key documents:

- `EN-GDP075FW1.pdf`: product/module specification for the GDP075FW1 family.
- `GDEM075F52.pdf`: Good Display 7.5 inch four-color panel specification.
- `JD79660A_datasheet_v1.0.3_20230620.pdf`: display controller datasheet, including the PTL partial-window register.
- `GDEY075T7.pdf`: related 7.5 inch mono panel reference used for comparison only.
